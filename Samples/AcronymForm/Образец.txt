2.1. PI Server
PI Server – это центральный компонент системы PI System, отвечающий за сбор, архивирование и распределение текущих данных. PI Server собирает все данные из различных источников, таких как корпоративные системы, базы данных и источники эксплуатационных данных, в единую систему. Он обеспечивает надежную защиту данных, предоставляя доступ на основе ролей, и предоставляет их пользователям на всех уровнях предприятия в единообразном, организованном виде.PI Server позволяет использовать минимальный объем дискового пространства для хранения данных, обеспечивая при этом необходимую точность и мгновенный доступ к данным вне зависимости от срока их хранения. Таким образом, пользователи получают наглядную и оперативную информацию об эксплуатационных показателях, ИТ-инфраструктуре и бизнес-операциях, что позволяет своевременно принимать обоснованные решения.
Масштабируемость и надежность являются одними из главных достоинств системы PI System. PI Server включает встроенную поддержку 64-разрядных технологий, многопроцессорного b и многоядерного оборудования и новейших операционных систем. Он обеспечивает высокую скорость сбора данных в режиме реального времени: операции сбора могут занимать менее секунды. Полученные данные мгновенно сохраняются на PI Server и становятся доступными пользователям в реальном времени. Кроме того, PI Server обеспечивает быстрый доступ к большим объемам данных вне зависимости от срока их хранения и позволяет хранить данные для миллионов точек. PI Server поддерживает горизонтальное и вертикальное масштабирование в зависимости от потребностей вашей организации. 
Встроенный компонент PI Asset Framework (AF) представляет нужные данные в соответствующем контексте и обеспечивает мгновенный доступ к ним на основе заданной модели ресурсов. PI AF позволяет организовывать, структурировать и искать данные PI и внешние данные в соответствии с различными типами объектов, например физическими объектами, задействованными в процессах (реакторы, трансформаторы, измерительные приборы и т.д.).
Эффективные инструменты расчета PI server позволяют осуществлять простые и настраиваемые вычисления, подсчитывать средние и итоговые значения и выполнять сложные программируемые расчеты KPI. Таким образом, пользователи PI System могут автоматически преобразовывать исходные данные в наглядную информацию, которую можно использовать для оперативного анализа, мониторинга и улучшения процессов, расчета KPI и т.д.
Встроенные функции оповещения позволяют внедрить логику мониторинга состояния в правила, которые используются в системе PI System для отслеживания данных в режиме реального времени и оповещения пользователей посредством различных каналов доставки о возникновении условий запуска. Пользователи могут самостоятельно настраивать содержимое этих уведомлений, которые могут включать различные типы информации, включая текущие данные, обзорную информацию о процессах, оптимальные методы, ссылки на веб-страницы и т.д. Используя необходимые данные и информацию, пользователи могут быстро реагировать на изменения в эксплуатационных условиях.
PI Server включает комплект PI System Management Tools (SMT), который представляет собой набор графических средств, инструментов командной строки с возможностью добавления сценариев и средств Microsoft® Excel® для управления и администрирования PI System. PI SMT предоставляет администраторам PI System возможность проверять архивы, управлять правами доступа пользователей, создавать теги, осуществлять поиск журналов сообщений, просматривать журнал аудита, контролировать состояние системы и выполнять множество других действий. Помимо этого, многие задачи, такие как обслуживание архивов и еженощное создание резервных копий Windows Volume Shadow-copy Services (VSS) могут выполняться в автоматическом режиме, что позволяет сократить объем работы системных администраторов. Комплект PI SMT предоставляет администраторам PI System мгновенный доступ к инструментам и информации, необходимой для обеспечения оптимальной работы системы.
PI Server поддерживает 9 типов данных, включая Int, Float, String, Blob и Timestamp, что позволяет осуществлять сбор данных в различных форматах.
PI Server использует встроенную систему безопасности Windows® и включает полную поддержку Active Directory™, что обеспечивает надежную защиту и мгновенный доступ к вашим данным посредством единого входа для всех пользователей. 
PI Server позволяет сохранить целостность данных благодаря регистрации всех внесенных в них изменений. Функция ведения журнала аудита соответствует самым строгим требованиям к электронной отчетности, включая стандарт FDA 21 CFR часть 11 (США), требования Агентства по охране окружающей среды (EPA) и других служб по надзору.
Пользователи из различных регионов могут получать доступ к данным PI System (вне зависимости от физического расположения PI Server или часового пояса, в котором находится источник данных) и автоматически просматривать их в местном часовом поясе.
Функция сжатия позволяет сэкономить объем хранящихся данных, предотвращая хранение повторяющихся или несущественных значений и повышая эффективность извлечения данных. Благодаря широким возможностям настройки пользователи могут самостоятельно решать, какие данные следует хранить.
Благодаря поддержке аннотаций PI Server позволяет пользователям добавлять заметки, комментарии, коды причины и дополнительную информацию к значениям для любых тегов PI.
2.2. PI Interface OPC DA
Интерфейс PI Interface для OPC DA — клиентское приложение OSIsoft, которое обеспечивает связь между сервером OPC и архивом PI Data Archive. Интерфейс PI Interface для OPC DA собирает данные в режиме реального времени из сервера OPC и записывает значения в точки PI Point. При необходимости интерфейс может записывать в сервер OPC данные из точек PI Point в архиве PI Data Archive. Интерфейс PI Interface для OPC DA построен на платформе OSIsoft UniInt, которая располагает стандартным набором типовых функций. Например, интерфейс может быть настроен на запуск в качестве автоматически запускаемой службы, запуск в отключенном состоянии, а также использование буферизации для предотвращения потери данных. Работа OPC зависит от служб Microsoft COM и DCOM, и интерфейс поддерживается только на платформах Windows. Интерфейс PI Interface для OPC DA поддерживает стандарт OPC Data Access версий 1.0a, 2.0 и 2.05.
2.3. PI DataLink
Клиентское приложение PI DataLink представляет собой дополнение для редактора таблиц Microsoft® Excel® и предназначено для предприятий, которые заинтересованы в повышении эффективности обработки информации и нуждаются в выполнении сложного анализа данных, создании моделей, составлении отчетов и агрегировании данных по процессам и бизнес-системам. С помощью приложения PI DataLink пользователи могут создавать гибкие отчеты, просматривать данные, поступающие в режиме реального времени, и производить вычисления в знакомой среде с использованием таблиц. В отличие от других продуктов, приложение PI DataLink разработано на основе ведущего в отрасли решения – PI Server, что обеспечивает доступ к большим объемам архивных и актуальных данных. 
PI DataLink устанавливает непосредственное соединение между OSIsoft PI System и Microsoft Excel. PI DataLink может отображать как архивные, так и данные в режиме реального времени,  хранящиеся на PI Server, в табличном редакторе Microsoft Excel, функциональность которого может использоваться для создания отчетов и выполнения сложного анализа данных. PI DataLink упрощает и ускоряет выполнение таких задач, как сбор информации, подготовка отчетов, моделирование, анализ, прогнозирование и планирование процессов.
С помощью PI DataLink пользователи могут получать максимальную экономическую выгоду от данных PI System. Располагая результатами анализа и обладая пониманием данных производственного процесса, имеется возможность вносить определенные изменения, направленные на улучшение качества, повышение степени соответствия, увеличение производительности и оптимизацию бизнес-процессов. 
Поскольку клиентское приложение PI DataLink построено на основе знакомой технологии, такой как Microsoft Excel и PI Server, специалисты могут использовать для работы уже имеющиеся у них знания и навыки, в результате чего снижается количество времени, необходимое на прохождение обучения использованию сложного нового инструмента.
